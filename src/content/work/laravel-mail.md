## Getting Started

1. Installation
```bash
composer require laravel-mail-platform
```

2. Configuration
```php
php artisan vendor:publish --provider="LaravelMail\ServiceProvider"
```

 Setup Workspaces
```php
php artisan mail:setup-workspace
```
